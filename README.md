# FullStack 3 - Actividad Sumativa 1

## Recetas del mundo

Sitio web desarrollado en Springboot + Thymeleaf para informar sobre recetas.

Está compuesto por las siguientes vistas:

* Página de inicio: (vista pública) Se muestran las recetas más recientes y populares. Incluye banners de avisos comerciales y acceso a la página Login.
* Login: (vista pública) Formulario para iniciar sesión.
* Detalle receta: (vista privada) Visualizar la receta, sus ingredientes, preparación y fotografía.

## Dependencias

* Springboot web
* Thymeleaf
* Spring security
* JWT
* Lombok
* Springboot Devtools

## Endpoint para ambiente de desarrollo
